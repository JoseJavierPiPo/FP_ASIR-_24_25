Definiciones

Charset

Hace referenecia a la codificación de caracteres utilizada en una página web.

Esta codificación determina cómo se representan los caracteres especiales, como acentos, tildes y símbolos

Autor

El autor se refiere a la persona o entidad que crea y publica el contenido de la página web

Description

Se utiliza para proporcionar una breve descripción del contenido de la página web.

Keywords

Las palabras clave se refieren a las palabras o frases clave que se utilizan para describir el contenido de una página web.
